# Carta

Carta es un menu digital con opciones de takeaway y delivery.

# Lenguajes y tecnologías

- NextJS
- Typescript
- Tailwind
- Shadcn
- Prisma
- (DB a definir)
- Zustand
- Zod
- React hot toast
- Cloudinary

# Funciones

- CRUD de categorías, subcategorías y productos.
- Realizar pedidos para takeaway y delivery.
